Kicker Speed Control

Precise idle control for kicker motors

Français | English

Key Features

  • High precision control of the servo angle (kicker speed)
  • Servo direction control (Clockwise or Counter-Clockwise)
  • If the application is restarted, it resumes from the last position
  • Stores the configured Max Angle in memory

Application (Android/App)

Control Buttons

  • Increase angle by 0.5 degree
  • Increase angle by 5 degrees
  • Decrease angle by 0.5 degree
  • Decrease angle by 5 degrees
  • IDLE – Return to idle speed
  • MAX SPEED – Full throttle (must be held)

GoByte Button

  • Provides a temporary speed increase or decrease for 10 seconds
  • Configurable cycles:
    • Every 2 minutes
    • Every 4 minutes
    • Every 6 minutes
    • Stronger speed increase or decrease every 4 minutes

SETUP Button

  • Set the servo rotation direction (performed once during installation)
  • Set the desired Max Angle (adjustable between 30 and 100 degrees)

KSC Control Box

  • Enclosed unit with display and rotary push encoder (water-resistant, splash-proof)

At Startup, the Display Shows:

  • Serial number
  • Bluetooth MAC address
  • Firmware version

During Operation, the Display Shows:

  • Servo angle (same value as shown in the application)
  • Selected GoByte mode (when active)
    • Remaining seconds before the next speed pulse

Encoder Button Functions:

  • Rotate the knob to adjust the servo angle (automatically updates the angle in the application)
  • Press the knob to return to idle speed
